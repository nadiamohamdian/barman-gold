# پنل ادمین بارمن گلد

پنل ادمین کامل و حرفه‌ای برای مدیریت فروشگاه طلا و جواهرات بارمن گلد

## ویژگی‌های اصلی

### 🏠 داشبورد
- آمار کلی فروشگاه (محصولات، سفارشات، کاربران، درآمد)
- آخرین سفارشات
- محصولات با موجودی کم
- عملیات سریع

### 📦 مدیریت محصولات
- لیست و جستجوی محصولات
- افزودن محصول جدید با محاسبه‌گر قیمت
- ویرایش و حذف محصولات
- مدیریت واریانت‌ها (وزن، عیار، موجودی)
- مدیریت دسته‌بندی‌ها و تگ‌ها
- آپلود تصاویر

### 🛒 مدیریت سفارشات
- لیست سفارشات با فیلتر
- تغییر وضعیت سفارشات
- مدیریت پرداخت‌ها
- پیگیری ارسال
- چاپ فاکتور

### 👥 مدیریت کاربران
- لیست کاربران
- مدیریت وضعیت کاربران
- ارسال ایمیل
- مسدود کردن/فعال کردن

### 💰 مدیریت قیمت‌ها
- اتصال به API قیمت طلا (BrsApi.ir)
- همگام‌سازی خودکار قیمت‌ها
- محاسبه‌گر قیمت محصول
- مدیریت قوانین قیمت‌گذاری
- حاشیه سود و مالیات

### 📝 مدیریت محتوا
- مدیریت مقالات
- مدیریت صفحات
- مدیریت منوها
- مدیریت بنرها

## تکنولوژی‌های استفاده شده

### Frontend
- **Next.js 15** - فریمورک React
- **TypeScript** - تایپ‌اسکریپت
- **Tailwind CSS** - استایل‌دهی
- **shadcn/ui** - کامپوننت‌های UI
- **Radix UI** - کامپوننت‌های پایه
- **React Hook Form** - مدیریت فرم‌ها
- **Zod** - اعتبارسنجی
- **TanStack Query** - مدیریت state و cache
- **TanStack Table** - جداول پیشرفته
- **Recharts** - نمودارها و آمار

### Backend
- **Node.js** - محیط اجرا
- **Express.js** - فریمورک وب
- **Prisma** - ORM دیتابیس
- **MySQL** - دیتابیس
- **JWT** - احراز هویت
- **Zod** - اعتبارسنجی

## API های استفاده شده

### قیمت طلا
- **BrsApi.ir** - دریافت قیمت لحظه‌ای طلا و ارز
- پشتیبانی از طلا ۱۸ و ۲۴ عیار
- پشتیبانی از سکه‌های طلا
- تاریخچه قیمت‌ها

### Endpoints اصلی
```
GET /v1/admin/dashboard          # داشبورد
GET /v1/admin/products          # لیست محصولات
POST /v1/admin/products         # افزودن محصول
PUT /v1/admin/products/:id      # ویرایش محصول
DELETE /v1/admin/products/:id   # حذف محصول
GET /v1/admin/orders            # لیست سفارشات
PUT /v1/admin/orders/:id/status # تغییر وضعیت سفارش
GET /v1/admin/users             # لیست کاربران
GET /v1/admin/price-feeds       # قیمت‌های طلا
POST /v1/admin/price-feeds/sync # همگام‌سازی قیمت‌ها
```

## نصب و راه‌اندازی

### 1. نصب پکیج‌ها
```bash
cd apps/web
pnpm install
```

### 2. تنظیم متغیرهای محیطی
```bash
# .env.local
NEXT_PUBLIC_BRS_API_KEY=your_api_key_here
NEXT_PUBLIC_API_URL=http://localhost:3001
```

### 3. اجرای پروژه
```bash
# Frontend
pnpm dev

# Backend (در ترمینال جداگانه)
cd apps/api
pnpm dev
```

### 4. دسترسی به پنل ادمین
```
http://localhost:3000/admin
```
